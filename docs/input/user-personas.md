### ランニング習慣

- **Garmin** で計測。
- 走行中は **Amazon Prime Music** や **radiko** をスマホで視聴。
- **ソロラン**が中心で、**マンネリ**を感じることがある。

### 課題・ペイン

- 音楽・一般ラジオは好きだが、**走る時間・距離・その日のシチュにピッタリ合う音声**という**選択肢**がもっと欲しい。
- 「**もっと走ったらどんな放送が聞けるか**」という**期待**がモチベになるが、既存サービスでは設計しにくい。

### 現在の解決方法

- Prime / radiko / Garmin の組み合わせ。不満というより**補完したい**。

### 求めているもの

- **好み・ゴール**（例：銭湯・ビール）に寄り添った**自然な声**のトーク。
- **セッション長に合った長さ**の「ラジオ」（例：30分ラン → 約30分）。
- **走り始める前**に生成が終わっている利用フロー。

### このプロダクトを使う瞬間

- 週次のジョグ前、**目標時間または距離**を決めたあと、**イヤホンをしながら走り出す直前**。

### 情報収集

- 同僚からの口コミ、社内デモ、（将来）SNS 等はスコープ外でもよい。
