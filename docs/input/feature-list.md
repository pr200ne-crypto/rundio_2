## 機能詳細（MVP コア）

### F-04 放送生成ジョブ（サーバー）

- **入力**: ユーザーID、プロフィール参照、目標時間または距離。
- **処理**: 台本生成（LLM）→ 音声合成（TTS）→ ファイル保存。
- **出力**: `status=ready`、再生用 URL または Storage パス。
- **制約**: API キーはサーバーのみ。長尺はタイムアウトに注意し非同期化を検討。

### F-05 音声再生

- iOS Safari の自動再生ポリシーに合わせ、**ユーザー操作後**に `play()`。
